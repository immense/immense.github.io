php Style Guide

Naming Conventions

Use Pascal case for classes. Camel case for class properties and methods. Underscores for normal variables and static method names.

Spacing and Braces

Set tabs to always be spaces, and use 4 spaces per indent.

Use the Allman style for curly braces for class and function definitions. The curly brace should be on the next line by itself.

Sample


        class TestThing
        {
            public $objectProperty;

            public function methodName()
            {
                ...
            } // End methodName

            public static function method_name() 
            {
                ...
            } // End method_name

        } // End TestThing

        function function_name( $argument = null ) 
        {
            ...
        } // End function_name

        $variable_name = new TestThing(); //Non-object and objects
